# [tinyapi](https://tinyapi.xyz)

**tinyapi** is a lightweight platform to quickly build and deploy serverless APIs.  

- Write small functions and run them in isolated containers  
- Persist and query data with built-in table storage (CRUD supported)  
- Test your API before deployment  
- Deploy with one click and get a shareable link:  https://tinyapi.xyz/api/{random-generated-token}
